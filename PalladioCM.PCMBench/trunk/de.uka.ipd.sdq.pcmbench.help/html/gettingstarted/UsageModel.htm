Creating a Usage Model

  1. In the Package Explor view, select the project and select File > New > Other... from the project´s context menu.

  2. Select Palladio Modeling > PCM Usage Model Diagram then click Next to start the wizard.

  3. On the next page click Finish.

  4. Right-click in the usage model diagram view and select Load Resource...

  5. Select Browse Workspace...

  6. In the File Selection-Window select your project and your system

  7. Click Ok and again Ok
