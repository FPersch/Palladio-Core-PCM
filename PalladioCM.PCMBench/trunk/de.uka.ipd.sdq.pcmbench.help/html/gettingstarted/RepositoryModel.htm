Creating a Repository Model

  1. In the Package Explor view, select the project. select New > Other... from the project´s context menu.

  2. Select Palladio Modeling > PCM Repository Model Diagram then click Next to start the wizard.

  3. On the next page click again Finish.

  4. Right-click in the repository diagram view and select Load Resource...

  5. Select Browse Workspace...

  6. In the File Selection-Window select your project and your resourcetype

  7. Click Ok and again Ok
