Creating a Repository Model

  1. In the Package Explor view, select the project. select New > Other... from the project´s context menu.
  2. Select Example EMF Model Creation Wizards > Repository Model then click Next to start the wizard.
  3. On the next page click again Next.
  4. Select the Model Object Repository and then click Finish.

Initialize Repository Diagram

  1. In the Package Explor view, make sure that your project is expanded so you can see your repository.
  2. Select Initialize repository_diagram diagram file from the project´s context menu.
  3. On the next page click  Next and then click Finish
