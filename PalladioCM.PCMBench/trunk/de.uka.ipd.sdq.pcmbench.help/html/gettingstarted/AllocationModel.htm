Creating a Allocation Model

  1. In the Package Explor view, select the project and select File > New > Other... from the project´s context menu.

  2. Select Palladio Modeling > PCM Allocation Diagram then click Next to start the wizard.

  3. On the next page click again  Next

  4. And click Next again.

  5. Then you get the response "No resource environment selected". CLick on nbsp;Change

  6. Click Browse Workspace...

  7. In the File Selection-Window select your project and your resourceenvironment

  8. Click Ok and again Ok

  9. Click Next

  10. And you get the response "No system selected". CLick on nbsp;Change

  11. Click Browse Workspace...

  12. In the File Selection-Window select your project and your system

  13. Click Ok and again Ok

  14. Click Finish
