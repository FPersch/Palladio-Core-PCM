repository:EventGroup class created and extends relatonship to Interface added
repository:EventGroup 1 <>-> * EventType added

repository:ProvidedRole -> Interface moved to ProvidedRole -> MethodInterface, navigatable in both directions, and aligned to naming conventions ("providedRoles__MethodInterface" instead of "providedRoles__Interface")
