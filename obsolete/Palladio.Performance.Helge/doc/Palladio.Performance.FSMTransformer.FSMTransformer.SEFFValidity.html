FSMTransformer.SEFFValidity Method 

Check wheter all states and all transitions have a QMLAttribute or a RandomVariable and whether the sum of probabilities of all outgoing transitions is 1. If sum > 1 an exception is thrown. If sum is smaller than 1, the user gets a warning. The remaining probability is shared to all transitions with no ProbabilityAttribute. All QMLAttributes are converted to RandomVariables.

A new IFiniteStateMachine where all states, except start state and final states, contain a RandomVariable and all transitions contain a RandomVariable and a ProbabilityAttribute.
