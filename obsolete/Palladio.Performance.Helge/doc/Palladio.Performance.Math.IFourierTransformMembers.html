IFourierTransform Members

IFourierTransform overview

Public Instance Methods

Backward Transforms a set of complex values from the frequency space to the time space.
BackwardToReal Transforms a set of complex values from the frequency space to the time space and converts the result to real values by neglecting the imaginary part.
Forward Transforms a set of complex values from the time space into the frequency space.
ForwardFromReal Transforms a set of real values from the time space into the (complex) frequency space.
