IDiscreteFunctionFactory.DiracDeltaFunction Method 

Returns the dirac function which is the neutral element of the convolution. This is: f(0) = infty and f(x) = 0 for x != 0.

IDiscreteFunction DiracDeltaFunction();

Return Value

Dirac function of length one. Function value is 1.
