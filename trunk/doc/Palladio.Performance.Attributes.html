Palladio.Performance.Attributes Namespace

Namespace hierarchy

Classes

Class Description
AttributeTransformer Class for transforming or generating Attributes. All methods are static.
FourierAttribute Attribute containing the fourier transformed probability density function.
MaximumExecutionTimeAttribute This attribute contains the maximum execution time for the probability density function of a regular expression. This attribute is needed for the calculation in frequency domain.
MeasureAttribute Attribute containing measured execution times.
PercentileAspect Class representing a percentile aspect of the QML.
ProbabilityAttribute Attribute containig the probability of a transition. Each transition in a service-effect-automaton should have an ProbabilityAttribute
QMLAttribute Describes the time behavior of a service by a list of percentile aspects of the Quality of Service Modeling Language. Usage: Percentile | Value 20% 20ms 40% 77ms 100% 100ms A percentile and a value together builds a qml aspect. The first line means that 20% of all execution of a service is below 20 ms.
RandomVariable Represents a random variable of a service-effect-transition or state. It defines the time behavior of a state or a transition in a service effect automaton.
