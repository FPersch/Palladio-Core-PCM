Public Instance Methods

Add Adds a function to this.
AddScaled Adds a function g scaled by a to the existing function.
AdjustSamplingRate Adjust the function to a new SamplingRate. The new Sampling rate must be a multiple or a divisor of the old one. The size of the data array is adjusted hold the values.
Convolution Convolution for discrete Functions. The two functions must have the same SamplingRate.
Crop Removes 0-Values at beginning and end.
DismissValuesSmalerThanN Eleminates values at beginning and end of the data array whose probability is smaller than n.
ExpandDomainOverloaded. Expands the domain of the function to the new value of xMax and xMin. If xMax or xMin are not a multiple of the samplingrate, they are set to the next multiple. All new values are set to zero.
ExpandDomainPo2 Expands the domain of the function to the new value of xMax and xMin. If xMax or xMin are not a multiple of the samplingrate, they are set to the next multiple. Furthermore additional values are added, so the size of the function becomes a power of 2. All new values are set to zero.
FunctionWithNValues Create a Function with n values. The sampling rate will be adjusted
GetScaled Returns a copy of the function scaled by the given factor.
MaximunDataValue Returns the maximum value of the function.
Multiply Multiplies this function with another function.
NumberOfNullValues Return the number of values which probability is 0.
Scale Scales this function by a.
Sum Returns the sum of all function values.
ToFile Writes the function to a file.
ValueAt Returns the time value (value on the x-axis) of the specified index.
ValueOf Returns the function value of x, that is f(x).
