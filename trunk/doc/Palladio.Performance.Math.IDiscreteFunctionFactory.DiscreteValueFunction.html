IDiscreteFunctionFactory.DiscreteValueFunction Method 

Creates a new function using a set of values. The sampling rate for those values must correspond to the sampling rate of the factory. Also the start value must be a multiple of the sampling rate.

IDiscreteFunction DiscreteValueFunction(
   long xStart,
   long samplingrate,
   double[] values
);

Parameters

xStart
Position on the x axis for the first value in the value array.
samplingrate
Distance of two values.
values
Ordered set of values describing the function.
