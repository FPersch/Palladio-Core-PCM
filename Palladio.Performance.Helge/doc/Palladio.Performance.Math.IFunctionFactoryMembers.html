IFunctionFactory Members

IFunctionFactory overview

Public Instance Methods

ConstantFunction Returns the constant zero function. This is: f(x) = c.
Copy Creates a copy of f with the parameters given by the factory.
DiracDeltaFunction Returns the dirac function which is the neutral element of the convolution. This is: f(0) = infty and f(x) = 0 for x != 0.
ExponentialDistribution Returns the exponential distribution with a rate. This is: f(x) = (1 / rate) * exp(- x (1 / rate)).
