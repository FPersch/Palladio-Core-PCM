IDiscreteFunction.ExpandDomain Method (Int64)

Expands the domain of the function to the new value of xMax. If xMax is not a multiple of the samplingrate, it is set to the next multiple. All new values are set to zero.

void ExpandDomain(
   long xMax
);

Parameters

xMax
New maximum x-value. If the value is smaller than the old xMax a ApplicationException is thrown.
