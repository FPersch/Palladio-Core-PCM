FSMTransformer.FSMWithIntermediateState Method 

Creates a new FSM where all Attributes holded by transitions. For each state s containing a QMLAttribute or RandomVariable a new intermediate state s' is created. The new transition s->s' holds the time consumption. For each state m with transition m->s, the transition is replaced by a transition m->s'.
