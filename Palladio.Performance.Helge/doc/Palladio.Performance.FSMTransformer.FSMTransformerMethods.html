CreateUniqueState Creates a state with a unique id.
FSMTimeConsumptionToString Creates a string representation of of the FSM including the time consumption.
FSMWithIntermediateState Creates a new FSM where all Attributes holded by transitions. For each state s containing a QMLAttribute or RandomVariable a new intermediate state s' is created. The new transition s->s' holds the time consumption. For each state m with transition m->s, the transition is replaced by a transition m->s'.
GetIncomingTransitions Returns all incoming transitions of a state.
SEFFValidity Check wheter all states and all transitions have a QMLAttribute or a RandomVariable and whether the sum of probabilities of all outgoing transitions is 1. If sum > 1 an exception is thrown. If sum is smaller than 1, the user gets a warning. The remaining probability is shared to all transitions with no ProbabilityAttribute. All QMLAttributes are converted to RandomVariables.
