Classes

Class Description
SeffWithAttributeCreator Class for creating the service effect automaton including the creation of attributes. The attributes are generated from from measures that are contained in a xml file.
WebserverModelHelper Class for creating states and transitions. All created states and transitions are accessible through a hashtable.
WebserverSeffFactory Factory for the creation of all required Seffs of the Webserver. This factory was created due to the missing save and load operations for the Seffs.
