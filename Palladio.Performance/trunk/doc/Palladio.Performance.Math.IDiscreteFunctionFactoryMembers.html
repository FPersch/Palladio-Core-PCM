IDiscreteFunctionFactory Members

IDiscreteFunctionFactory overview

Public Instance Properties

SamplingRate Precision for the created functions.
XMax Default maximum for the greatest x value that has an associated function value.
XMin Default minimum for the smallest x value that has an associated function value.

Public Instance Methods

ConstantFunction Returns the constant function. This is: f(x) = c.
Copy Creates a copy of a IDiscreteFunction with the parameters given by the factory.
DiracDeltaFunction Returns the dirac function which is the neutral element of the convolution. This is: f(0) = infty and f(x) = 0 for x != 0.
DiscreteFourierFunctionOverloaded. Creates a new fourier function.
DiscreteValueFunction Creates a new function using a set of values. The sampling rate for those values must correspond to the sampling rate of the factory. Also the start value must be a multiple of the sampling rate.
DiscreteValuePDFunctionOverloaded. Creates a new probability density function using a set of values.
ExponentialDistributionOverloaded. Returns the exponential distribution with a rate. This is: f(x) = (1 / rate) * exp(- x (1 / rate)).
